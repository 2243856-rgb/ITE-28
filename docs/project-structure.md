# Project Structure (Starter)

## Root

- `backend/` - Express API starter
- `database/` - SQL schema and future migrations
- `docs/` - implementation and architecture notes

## backend/

- `src/index.js` - base API app and health endpoint
- `.env.example` - local environment variables template
- `package.json` - scripts and dependencies

## database/

- `schema.sql` - initial SQL Server schema for MVP modules
